# Module licensing

PunishBridge uses separate adapter artifacts so provider-license obligations are visible.

| Module | License |
| --- | --- |
| `punishbridge-api`, `punishbridge-paper`, `punishbridge-testkit`, `punishbridge-provider-litebans` | Apache-2.0 |
| `punishbridge-provider-advancedban`, `punishbridge-provider-essentialsx` | GPL-3.0 |
| `punishbridge-provider-libertybans` | AGPL-3.0 |

The LiteBans API is MIT licensed; its notice is retained in [LiteBans-MIT.txt](LiteBans-MIT.txt).
Provider APIs are compile-only and are not bundled in PunishBridge artifacts. Embedding a GPL or AGPL adapter may affect the license obligations of the consuming plugin. Review the relevant license before distribution:

- [GPL-3.0](https://www.gnu.org/licenses/gpl-3.0.txt)
- [AGPL-3.0](https://www.gnu.org/licenses/agpl-3.0.txt)

This file describes the project layout and is not legal advice.
